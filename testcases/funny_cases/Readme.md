## Test Cases for Fun

This directory includes many funny test cases, such as maze、spiral、marvel characters、hilbert curve, etc.

You can upload these test cases to the [simple polygon generation web site](https://equation314.github.io/simple-polygon/#) and test the efficiency and correctness of  the triangulation and shorest path algorithm.

#### Details

- The number contained in the file name indicates the size of the point set.
- You can use the scripts in `generation_scripts` to create more points.
